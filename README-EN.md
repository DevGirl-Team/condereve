## Requirements

This repo use [nextjs][1].

Minimum Requirements:

- [nodejs][2] ^14.13.1 || >=16.0.0
- [yarn][3] >=1.22.0

## Installation

1. Clone `git@github.com:DevGirl-Team/ConDeReve.git`
2. ```bash
   yarn && cd front/ && yarn
   ```

## Use

```bash
cd front/
yarn dev # point to localhost:3000
```
