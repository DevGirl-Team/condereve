## Requis

Ce repo utilise [nextjs][1].

Le minimun requis:

- [nodejs][2] ^14.13.1 || >=16.0.0
- [yarn][3] >=1.22.0

## Installation

1. Cloner le repo `git@github.com:DevGirl-Team/ConDeReve.git`
2. ```bash
   yarn && cd front/ && yarn
   ```

## Utilisation

### Environement de dev

```bash
cd front/
yarn dev # pointe sur localhost:3000
```
